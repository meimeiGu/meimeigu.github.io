##React是什么

构建用户界面的JavaScript库

为数据提供渲染为HTML视图的JavaScript库

上面是官网和维基百科给出的定义，可以看出来React是围绕着渲染界面为重心。它有哪些特点或者说优势呢：

在React中，React通过elements来描述组件实例或者DOM节点，elements是一个对象，包含了描述对象的类型、属性和里面的任何子元素，这正是我们常常听到的virtual elements，通过操作virtual elements来管理整个virtual DOM。React elements分为两种类型，描述DOM节点或者组件这两种类型的react元素，通过tag区分，如果为字符串则是DOM元素，为类或者函数则是组件元素。当React解析发现tag为组件元素时，会向该组件询问组件要返回什么，一直到拿到的都是dom元素。

我们的JSX，是一个javaScript语法扩展，JSX编译器会将写的节点解析成React elements树，大家可以去试着将这个对象转化为真正的dom结构。

React里的组件，从概念上类似于JavaScript函数。它接受任意的入参（即“props”）,并返回用于描述页面展示内容的React元素。
